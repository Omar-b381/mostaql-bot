## 📌 وصف المشروع

**Mostaql Project Hunter** هو بوت أتمتة مبني بـ Python يعمل بشكل دوري على مراقبة منصة مستقل — أكبر منصة عمل حر في العالم العربي — بحثاً عن مشاريع في مجال تحليل البيانات. عند اكتشاف مشروع جديد مطابق، يقوم البوت تلقائياً بـ:

- استخراج وصف المشروع الكامل
- تحليله بالذكاء الاصطناعي (Gemini) لاستخلاص المتطلبات الأساسية
- كتابة مسودة عرض (Proposal) احترافية جاهزة للإرسال
- إرسال كل هذا فورياً عبر تيليجرام

---

## 🎯 المشكلة التي يحلها

يواجه المستقلون في مجال تحليل البيانات تحديين رئيسيين:

| التحدي | الواقع |
|---|---|
| **السرعة** | المشاريع الجيدة تحصل على عروض خلال دقائق من نشرها |
| **الجهد** | كتابة عرض مخصص لكل مشروع تستهلك وقتاً وطاقة |

البوت يحل المشكلتين معاً: **يرصد المشاريع فور نشرها** ويقدم **مسودة عرض جاهزة** في ثوانٍ.

## 💡 الدافع وراء المشروع

كمحلل بيانات مستقل، لاحظت أن الفرصة الحقيقية في منصة مستقل تُحسم في أول 15 دقيقة من نشر أي مشروع. الفكرة كانت بسيطة: **لماذا أراقب المنصة يدوياً وأنا أستطيع بناء بوت يفعل ذلك بدلاً عني؟** هذا المشروع هو تجسيد عملي لفكرة استخدام الأتمتة والذكاء الاصطناعي لتعزيز الإنتاجية الشخصية في العمل الحر.

## 🛠️ التقنيات المستخدمة

| التقنية | الدور | سبب الاختيار |
|---|---|---|
| `Python 3.10+` | لغة البرمجة الأساسية | سرعة التطوير وثراء المكتبات |
| `requests` | إرسال HTTP requests | بسيطة وموثوقة لجلب صفحات الويب |
| `BeautifulSoup4` | تحليل HTML واستخراج البيانات | سهولة الاستخدام مع صفحات HTML المعقدة |
| `google-genai` | تحليل النصوص بالذكاء الاصطناعي | Gemini 2.0 Flash: سريع، دقيق، ومجاني نسبياً |
| `Telegram Bot API` | إرسال الإشعارات | الأسرع والأبسط للإشعارات الفورية على الموبايل |
| `os` + `txt file` | تتبع المشاريع المُرسلة | حل خفيف بدون Overhead قاعدة بيانات |
